# bamazon- Inventory tracking with mysql and node

## Requirements
  * node
  * cli-table
  * inquirer
  * mysql
  
 ## Instructions
 
App will give prompts for making purchases, purchases will subtract inventory from mysql table. Process loops until user is finished making purchases.
